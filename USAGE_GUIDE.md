# Claude Code 监控工具 - 使用指南

## 快速开始

### 1. 安装依赖
```batch
install.bat
```

### 2. 运行程序
```batch
python monitor_tool.py
```

## 完整使用流程

### 步骤 1: 选择监控区域

1. 点击"选择区域"按钮
2. 程序窗口会隐藏，屏幕变为半透明
3. **拖动鼠标**画出要监控的矩形区域
4. 松开鼠标后，会显示黄色确认框
5. **按 Enter 键**确认区域（或按 ESC 取消）

### 步骤 2: 选择点击位置

1. 区域确认后，屏幕继续半透明状态
2. 提示：**"现在请点击输入框的位置"**
3. **点击**Claude Code 输入框的位置
4. 会显示红色十字标记
5. **再次按 Enter 键**确认（或按 ESC 取消）

### 步骤 3: 查看结果

1. 程序窗口恢复显示
2. 屏幕上会显示**红色边框**标记监控区域
3. 弹出确认对话框显示详细信息
4. 播放系统提示音（Windows）

### 步骤 4: 调整参数

#### 检查间隔
- **范围**: 10-60 秒
- **默认**: 30 秒
- **说明**: 每隔多久截图检查一次画面变化

#### 触发时间
- **范围**: 30-120 秒
- **默认**: 45 秒
- **说明**: 画面无变化多久后触发自动操作

#### 相似度阈值
- **范围**: 0.90-0.99
- **默认**: 0.98
- **说明**:
  - 越高越严格（0.98推荐）
  - 越低越敏感（0.95更敏感）
  - 值越高 = 需要画面更相似才判定为"无变化"
  - 值越低 = 微小变化也能触发操作

#### 记忆位置
- **选项**: 复选框
- **默认**: 勾选
- **说明**:
  - 勾选：下次启动自动恢复区域和点击位置
  - 不勾选：每次启动需要重新选择

#### 发送消息
- **默认**: "Go ahead with ur todo list. Remeber to update the PROJECT_PLAN.MD and essestials files, stick to the Claude.md file and git push for significant files changes."
- **说明**: 触发时自动发送的文字（支持中文）

### 步骤 5: 开始监控

1. 点击"开始监控"按钮
2. 弹出确认对话框，显示所有参数
3. 点击确定开始监控
4. 状态栏显示"监控运行中..."

### 步骤 6: 自动化工作

监控运行时：
1. 每隔 N 秒截取监控区域的图像
2. 与上一次截图对比相似度
3. 如果持续 M 秒无变化（相似度 >= 阈值）：
   - 点击指定的输入框位置
   - 将消息复制到剪贴板
   - 按 Ctrl+V 粘贴
   - 按 Enter 发送
4. 继续监控，循环执行

### 步骤 7: 停止监控

- 点击"停止监控"按钮即可

## 进阶功能

### 清除配置

**使用场景**:
- 想要重新选择区域
- 红色边框显示错误
- 配置文件损坏
- 恢复默认设置

**操作步骤**:
1. 点击"清除配置"按钮
2. 在确认对话框中点击"是"
3. 所有设置恢复默认值
4. 红色边框消失
5. 配置文件被删除

### 调整记忆位置设置

**启用记忆**（推荐）:
- 勾选"下次启动时自动使用本次框选和点选位置"
- 下次启动自动显示红色边框
- 无需重新选择区域

**禁用记忆**:
- 取消勾选复选框
- 下次启动需要重新选择区域
- 适合每次监控不同位置的情况

## 实用技巧

### 1. 紧急停止
- **鼠标移到屏幕角落**可紧急停止自动化操作
- 这是 pyautogui 的安全机制

### 2. 取消选择
- 在选择区域或点击位置时，按 **ESC** 键可以取消操作

### 3. 调整阈值
- 如果总是触发（画面有变化却判定无变化）→ **降低阈值**（如 0.95）
- 如果不触发（画面无变化却不执行）→ **提高阈值**（如 0.99）

### 4. 最佳实践
- **检查间隔**: 30 秒 （平衡性能和响应速度）
- **触发时间**: 45-60 秒（避免误触发）
- **相似度阈值**: 0.98（适合大多数场景）

### 5. 测试建议
1. 首次使用建议白天测试几次
2. 观察红色边框是否正确覆盖监控区域
3. 手动让 Claude Code 停止一段时间，看是否自动触发
4. 确认消息能正确输入（中文显示正常）

## 常见问题

### Q: 红色边框不见了怎么办？
A:
1. 检查"记忆位置"是否勾选
2. 点击"选择区域"重新选择
3. 或点击"清除配置"后重新配置

### Q: 中文输入不显示？
A:
- 程序已使用 Ctrl+V 粘贴方式，支持中文
- 确保 `pyperclip` 已安装（在 requirements.txt 中）

### Q: 监控不触发？
A:
1. 检查阈值是否太高（试试 0.95）
2. 检查触发时间是否太长
3. 查看状态栏是否显示"监控运行中"
4. 确认监控区域内确实无变化

### Q: 总是误触发？
A:
1. 提高阈值（试试 0.99）
2. 增加触发时间（如 60 秒）
3. 确保监控区域内没有动画或闪烁元素

### Q: 程序启动后配置丢失？
A:
- 检查"记忆位置"复选框是否勾选
- 查看是否存在 `monitor_config.json` 文件
- 确认上次关闭前点击了"开始监控"（会保存配置）

```json
  "remember_position": true,

}
```
